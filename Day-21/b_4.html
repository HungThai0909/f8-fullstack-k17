Cho trước mảng sau:

const posts = [ { id: 1, title: "JavaScript cơ bản", tags: ["js", "basic"], comments: [ { user: "An", text: "Hay quá!" }, { user: "Bình", text: "Rất dễ hiểu" }, ], }, { id: 2, title: "Học React không khó", tags: ["react", "js"], comments: [{ user: "Chi", text: "Cảm ơn chia sẻ" }], }, ];

Yêu cầu:

  • In ra tất cả title kèm số lượng comments của từng bài viết.
  • Tạo mảng mới chứa tất cả tags (không trùng lặp).
  • Tìm tất cả các bình luận của user "An".
